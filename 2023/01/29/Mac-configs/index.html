Mac configs

Basic Mac configurations

输入设备

工欲善其事，必先利其器。

键盘：

  • 键重复速率：快

  • 重复前延迟：短

  • 按下地球仪键时：不执行任何操作

  • 键盘快捷键...

    • 启动台与程序坞：

      • 启用显示启动台 on Control + Command + E

    • 调度中心：

      • 修改调度中心 on Control + Command + Tab

      • 修改显示桌面 on Control + Command + W

    • 聚焦：

      • 修改显示App on Control + Command + E

    • App快捷键：

      • 为所有应用程序添加启动终端 on Control + Command + T

      • 为所有应用程序添加启动浏览器 on Control + Command + S

      • 为所有应用程序添加一时活跃 on Control + Command + A

    • 功能键：将F1、F2等键用作标准功能键：启用

    • 修饰键：Capslock to Control && Control to Capslock

  • 文字输入 → 输入法 → 编辑... → 所有输入法

    • 自动纠正拼写：禁用

    • 自动大写字词的首字母：禁用

鼠标：

  • 跟踪速度：5格

  • 高级...

    • 指针加速：禁用

触控板：

  • 跟踪速度：5格

  • 触控板点按：弱

  • 轻点来点按：启用

  • 系统设置 → 辅助功能：动作 → 指针控制 → 触控板选项...

    • 使用触控板进行拖移：启用

    • 拖移样式：三指拖移

通用

无丝竹之乱耳，无案牍之劳形。

系统设置 → 通用：

  • 本机名称：Cynthia

  • iMessage转发：iPhone Settings → Messages → Text Message Forwarding

    • 可能需要手动登陆：设置 → 信息 → 发送与接收 → 使用AppleID登陆iMessage信息

系统设置 → 辅助功能：

  • 显示：摇动鼠标指针以定位 → 禁用

  • 阅读与朗读：系统声音 → 月（高音质）

系统设置 → 外观：

  • 图标与小组件样式：深色

  • 显示滚动条：滚动时

系统设置 → 桌面与程序坞：

  • 自动隐藏和显示程序坞：启用

系统设置 → 菜单栏：

  • 电池：电池选项... → 显示百分比 → 启用

系统设置 → 声音：

  • 提醒声音：Crystal

  • 启动时播放声音：禁用

访达：

  • 设置：开启新“访达”窗口时打开：~ → 关闭窗口 → 新建“访达”窗口

  • 显示视图：为列表 → 用作默认

照片：

  • 选择的照片：右键 → 共享... → 设定墙纸

词典：

  • 设置：调整常用的参考源（重启生效）
