通用：

  • 本机名称：Camellia

  • iMessage转发：iPhone Settings → Messages → Text Message Forwarding

键盘：

  • 键盘重复率：快

  • 重复前延迟：短

  • 按下地球仪键时：不执行任何操作

  • 功能键：将F1、F2等键用作标准功能键：启用

  • 修饰键：Capslock to Control

触控板：

  • 触控板点按：弱

  • 轻点来点按：启用

  • 使用触控板进行拖移：启用

  • 拖移样式：三指拖移

访达：

  • 设置：开启新“访达”窗口时打开：~ → 关闭窗口 → 新建“访达”窗口

  • 显示选项：排序方式：名称 → 用作默认

  • 显示视图：为分栏 → 用作默认
