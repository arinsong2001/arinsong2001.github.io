在zsh中上述定义会报错，因为repeat是zsh的保留字。

repeat也是一个内置命令，用法：repeat n command, n为要执行的次数。

自定义函数可以覆盖shell的同名内置命令，但不会覆盖保留字。

本例，函数定义前加上关键字function就不会报错，但仍不会生效。

不使用function可以减少自己定义的函数与shell保留字重名的可能。
